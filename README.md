# Morph Shape Visualizer

Herramienta interactiva para explorar formas 2D con aristas curvas y perspectiva de rotación 3D.

## Demo

→ [Ver en GitHub Pages](https://agustingomila.github.io/morph-shape/)

## Controles

| Control        | Descripción                                                       |
|----------------|-------------------------------------------------------------------|
| Ángulo de giro | Rotación en perspectiva (0°–360°)                                 |
| Lados          | Número de vértices (1 = círculo, 2 = cápsula, 3–12 = polígono)    |
| Radio esquina  | Redondez de los vértices                                          |
| Curva arista   | Curvatura de las aristas (negativo = cóncavo, positivo = convexo) |
| Grosor canto   | Anchura del canto lateral                                         |
| Rotación forma | Orientación de la forma en el plano                               |
